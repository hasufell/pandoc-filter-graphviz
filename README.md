# pandoc-filter-graphviz

This repository provide a filter to [pandoc](http://pandoc.org/).
This filter process a specific code block marker `graphviz` in order to produce a pandoc output with a gererated picture.

'~~~ graphviz

Lorem ipsupm ....

~~~'
